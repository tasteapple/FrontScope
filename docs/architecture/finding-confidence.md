# Finding Confidence / finding 신뢰도

## Purpose / 목적

This layer helps readers distinguish between heuristic signals, matched evidence, and validated observations.
이 계층은 독자가 휴리스틱 시그널, 매칭된 근거, 검증된 관찰을 구분할 수 있게 돕는다.

## Confidence Levels / 신뢰도 수준

### `heuristic`
Pattern-based signal that suggests review is warranted.
패턴 기반 시그널로서 추가 검토가 필요함을 의미한다.

### `matched`
A finding backed by a stronger direct match, version mapping, or explicit response evidence.
더 직접적인 매칭, 버전 매핑, 명시적 응답 근거가 있는 finding이다.

### `validated`
A finding confirmed through direct response inspection or successful follow-up validation.
직접 응답 확인 또는 후속 검증 성공으로 확인된 finding이다.

## Why It Matters / 왜 중요한가

As FrontScope grows, not all findings should be read with the same certainty.
FrontScope가 커질수록 모든 finding을 같은 확실성으로 읽으면 안 된다.

Confidence labels help prioritize review and reduce confusion between weak heuristics and stronger evidence-backed observations.
신뢰도 라벨은 검토 우선순위를 정하고 약한 휴리스틱과 강한 근거 기반 관찰을 구분하는 데 도움이 된다.
